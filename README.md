# Titanic Survival Prediction

## Overview
A classification project to predict which passengers survived the Titanic disaster using machine learning models.

## Dataset
- **Source:** Kaggle — Titanic Dataset
- **Size:** 891 passengers, 12 variables
- **Target variable:** Survived (0 = No, 1 = Yes)

## Models Used
| Model | Technique |
|---|---|
| Logistic Regression | Binary classification |
| Decision Tree | Tree-based classification |

## Pipeline Steps
1. Data cleaning — handling missing values
2. Feature selection and encoding
3. Train/test split
4. Model training and evaluation
5. Cross-validation (10-fold)

## Technologies
- Python
- pandas, numpy
- scikit-learn (DecisionTreeClassifier, LogisticRegression, cross_val_score)
